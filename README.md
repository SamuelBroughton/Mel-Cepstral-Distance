# Mel-Cepstral Distortion

This repo is concerned with finding the Mel Cepstral Distortion between `.wav` files.

# Dependencies

* librosa
* mcd

# Installation

**Tested on a Python verison 3.6.2 in linux VM environment**

## Python packages

```
conda install -c conda-forge librosa
pip install mcd==0.4
```
